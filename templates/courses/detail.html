Курсы PyBursa

Description

Практический курс по изучению современного языка программирования Python и фреимворка Django. Обучение будет проходить с уклоном в веб-разработку. Мы учим начинающих программистов до уровня Junior Python Developer и после окончания курса все ученики могут претендовать на эту должность в любой интернет-компанию. Весь курс построен на практике: преподаватели-практики, обучение на примере разработки реальных проектов, практические домашние задания. На курсе мы Вас познакомим с языком Python, фреимворком Django, научим работать с базами данных, покажем, как проводить UnitTest, вместе сделаем работающий проект на Python / Django и освоим еще много полезного. После успешного окончания обучения всем предоставляется именной сертификат, который высылается почтой владельцу. Для нас это уже третья группа, и многие из наших учеников успешно работают в сфере Python Development. Курс совершенствуется из года в год и становится еще более эффективным.
